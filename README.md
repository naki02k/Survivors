# Survivors

## 制作者
神崎 健太

## 制作期間
2024年10月～2025年1月（約4か月）

## 概要

個人制作につき、企画・デザイン・プログラムを含むすべての工程を担当しました。
プログラム面では、Unityの便利な機能だけに依存するのではなく、描画やメモリ管理の仕組みを意識し、可読性・保守性・実行速度を重視した設計を心掛けました。

# プログラムのポイント

## Player.cs
### プレイヤー制御とクラス設計
- カプセル化を徹底（private / Property / private set）
- Early Returnを活用し可読性を向上
- 他クラスからの意図しない変更を防止
### 自動攻撃の最適化
- Coroutineで一定間隔攻撃を実装
- WaitForSecondsをキャッシュしGC Allocを削減

## ShopManager.cs
### ショップシステム
- IsShopOpenでゲーム状態を管理
- ショップ終了後3秒のカウントダウンを実装
- 重複しないランダム抽選
- Nullチェックによる堅牢性向上

## EnemyManager.cs
### 敵出現管理
- ビットシフトを利用した指数関数的Wave設計
- Mathf.Minで最大生成数を制限
- sqrMagnitudeによる高速距離判定

## WaveManager.cs
### ゲーム進行制御
- Coroutineによるゲームシーケンス管理
- Wave進行に応じた難易度調整
- Mathf.Maxによる安全な出現制御
